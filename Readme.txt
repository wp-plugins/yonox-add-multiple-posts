Yonox Add Multiple Posts enables a WordPress user to quickly add multiple posts or pages at once.

== Description ==

* Specify Post Type content to be added is a post or a page.
* If Pages are added, user can specify Parent Pages (to add children).
* User can specify the Author of new posts/pages.
* User can specify post status (Published / Draft / Pending Review).
* User can now choose a category from a drop-down to assign to new posts.
* Support translation for Spanish and Romanian languages.

This plugin is designed to:

* Quickly inserting multiple posts or pages at once to be edited later (with Post or Page Status option).
* Perform a new site with the desired pages and sub-pages.

== Installation ==

1. Extract the zip file and just drop the contents in the wp-content/plugins/ directory of your WordPress installation.
2. Activate the Plugin from Plugins page.
3. Go to the new admin menu created.
4. That's all folks. Enjoy.
